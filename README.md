Bienvenue sur mon portfolio personnel.  
Je suis étudiant en ingénierie informatique et technicien support en Télécoms, Réseaux et Informatique.  
Ce portfolio présente mes projets, compétences, expériences et réalisations académiques et professionnelles.

Un seul fichier index.html — HTML + CSS + JavaScript — hébergé sur GitHub Pages.

## 📂 Projets

### 🔹 Hemolink
Plateforme intelligente de don de sang intégrant l’IA et la géolocalisation.

### 🔹 TranspoBot
Application de gestion de flotte avec chatbot IA Text-to-SQL.

### 🔹 Réseau Multi-AS
Projet réseau utilisant les protocoles BGP, OSPF et EIGRP dans une architecture multi-AS.
